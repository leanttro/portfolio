Projetos em Destaque

Dashflix - Análise do Catálogo da Netflix

Processo completo de ETL e visualização de dados do catálogo da Netflix, resultando em um dashboard interativo para explorar insights sobre o conteúdo da plataforma.

Visão Geral do Projeto

O projeto é dividido em duas etapas: um processo de ETL (Extração, Transformação e Carga) para limpeza e preparação dos dados, e a criação de um dashboard em Power BI para visualização.

Processamento e Limpeza de Dados (ETL)

  • Os dados brutos são lidos e, utilizando Python e Pandas, são limpos e duplicatas removidas.
  • O pipeline de dados é estruturado em camadas (Bronze, Prata e Ouro) e armazenado em um banco SQLite (simulando um lakehouse).

Visualização de Dados em Power BI

O arquivo final limpo é usado como fonte para o dashboard "Dashflix", que permite explorar métricas como contagem de filmes vs. séries, distribuição por país e tendências ao longo dos anos.

Tecnologias Utilizadas

Python, Pandas, SQLAlchemy & SQLite, Power BI, e Google Colab.

Código Fonte

Pipeline VLE - Open University - Student Analysis

Análise completa sobre o engajamento e desempenho dos alunos no Ambiente Virtual de Aprendizagem (VLE) para prever a probabilidade de aprovação.

Objetivo do Projeto

Explorar dados demográficos, analisar a interação dos alunos com a plataforma, identificar fatores que influenciam o sucesso acadêmico e construir um modelo de Machine Learning para prever o resultado final do aluno.

Principais Insights

  • Alunos de regiões com mais de 50% de pobreza tiveram mais aprovados que reprovados.
  • A atividade (cliques) aumenta significativamente perto de avaliações, especialmente entre os aprovados.
  • A região da Irlanda, com menor índice de pobreza, apresentou a maior taxa de aprovação.

Tecnologias Utilizadas

Python (Pandas, NumPy, Matplotlib, Seaborn), Scikit-learn para modelagem de Machine Learning (Random Forest), e SQLAlchemy para simulação de armazenamento em lakehouse.

Análise de Custos de Infraestrutura (SICRO 2022-2024)

Análise exploratória de dados públicos do SICRO, focada na limpeza, tratamento e visualização dos custos de insumos para obras de infraestrutura no Brasil.

Objetivo do Projeto

O objetivo principal é extrair, tratar e analisar os dados de custos de composições e insumos do SICRO, fornecendo uma visão clara sobre as variações de preços de materiais e serviços essenciais para a infraestrutura.

Metodologia e Tecnologias

  • Coleta e Extração: Dados brutos em formato .xlsx do portal do DNIT.
  • Tratamento e Limpeza: Uso da biblioteca Pandas em Python para limpeza, remoção de nulos e padronização.
  • Análise Exploratória (EDA): Cálculo de estatísticas descritivas para identificar tendências e outliers.
  • Visualização de Dados: Uso de Matplotlib e Seaborn para criar gráficos ilustrativos.
